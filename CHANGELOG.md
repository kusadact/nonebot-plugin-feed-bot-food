# CHANGELOG

本文件只记录插件本身的功能更新与 bugfix。

## [0.1.0] - 2026-07-13

### Added

- 增加 OneBot 群聊投喂和体重状态查询。
- 增加基于 OpenAI 兼容接口的食物分类与增重判断。
- 增加固定窗口分类额度、LLM 请求硬上限和窗口边界槽位顺延。
- 增加 JSON 持久化、每日 06:00 减重和离线结算补偿。
- 增加可选的 groupmate-agent `feed_bot_food` 与 `get_feed_bot_status` Tools。

### Fixed

- 调整结果提示：无法确认分类时返回未投喂提示，LLM 请求或分类失败时保持静默。
- 明确 Agent Tool 对不可食用结果返回用户提示，仅分类或 LLM 请求失败静默处理。
- 明确 LLM 对多个食物输入只按主要类别计算一次增重，不拆分为多个投喂。
- 防止 LLM HTTP 错误响应正文泄漏 API Key。
- 将 LLM 3xx 响应按请求失败处理。
- 修正多槽位额度的下一次可用时间计算。
- 允许空投喂命令返回参数提示。
- 将直接投喂和状态查询改为标准 `/投喂`、`/查看体重` 命令，并增加 `/查看状态` 别名；不再要求艾特 Bot。
